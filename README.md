# SynapseVisionLab - EEG Data Processor
# Overview
SynapseVisionLab is a cross-platform desktop application for viewing, processing, and analyzing electroencephalography (EEG) data. Built with C++ and Qt, it provides an intuitive interface for visualizing multi-channel EEG signals and applying various signal processing techniques.

# Features
Multi-format Support: Load and save EDF, CSV, TXT, and DAT files

Interactive Visualization: Scroll, zoom, and pan through EEG recordings

Channel Management: Select/deselect channels, highlight individual channels

Signal Processing:

Bandpass, highpass, lowpass, and notch filters

Gain and offset adjustment

DC offset removal

Signal normalization

EEG Montages: Bipolar, average reference, and Laplacian montages

Statistics: View channel statistics (mean, std dev, min, max)

Cross-Platform: Runs on macOS and Windows
